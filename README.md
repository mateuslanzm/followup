# Follow-up Python

Repositório criado para compartilhar conteúdo sobre Python no LinkedIn.

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| [`followup_python.ipynb`](followup_python.ipynb) | Jupyter Notebook com os principais conceitos de Python |
| [`index.html`](index.html) | Página HTML de apresentação do conteúdo |

## Tópicos abordados

- Variáveis e tipos de dados
- Estruturas de dados (list, tuple, dict, set)
- Controle de fluxo (if/else, for, while, list comprehension)
- Funções (def, lambda, map, filter, recursão)
- Programação Orientada a Objetos (classes, herança)
- Tratamento de exceções
- Leitura e escrita de arquivos / JSON
- Bibliotecas padrão (datetime, random, statistics)

## Como executar o notebook

```bash
# Instalar o Jupyter se necessário
pip install notebook

# Abrir o notebook
jupyter notebook followup_python.ipynb
```
